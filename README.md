=======
# 🛒 E-Commerce Website

A full-stack e-commerce web application built using **React (Vite)** for the frontend and **Spring Boot** for the backend.
This project demonstrates a modern architecture for scalable online shopping platforms.

---

## 🚀 Tech Stack

### 🖥️ Frontend

* React + Vite
* JavaScript (ES6+)
* HTML5 & CSS3
* Fetch API

### ⚙️ Backend

* Spring Boot
* REST APIs
* Maven
* Java 17+

### 🗄️ Database

* MySQL 

---

## ✨ Features

### 🛍️ User Side

* Browse products
* View product details
* Add to cart
* Checkout flow (planned)

### 🔧 Backend APIs

* Product listing API
* Product management (CRUD — planned)
* User authentication (planned)
* Order management (planned)

---

## 🖥️ Frontend Setup (React + Vite)

### Install Dependencies

```bash
cd frontend
npm install
```

### Run Frontend

```bash
npm run dev
```

Frontend will run on:

```
http://localhost:5173
```
